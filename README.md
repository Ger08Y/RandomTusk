# Random Task Generator - Генератор случайных задач 🎲

**Автор:** Герман Яковлев
**Дата сдачи** 06.05.2026

## Описание

Random Task Generator - это удобное GUI-приложение для генерации случайных задач из разных категорий (Учёба, Спорт, Работа). Приложение позволяет фильтровать задачи по типу, добавлять свои задачи, сохранять историю генерации и просматривать статистику.

### Основные возможности:

- ✅ Генерация случайных задач из предустановленного списка
- ✅ Фильтрация по типу задачи (Учёба/Спорт/Работа/Все)
- ✅ История всех сгенерированных задач с датой и временем
- ✅ Сохранение истории в JSON и загрузка при запуске
- ✅ Добавление своих задач (с проверкой на пустую строку)
- ✅ Очистка истории
- ✅ Статистика по типу задач и дням
- ✅ Валидация ввода при добавлении новых задач
